# MarkEdit 文件类型支持功能展示

## 🎯 新增功能概览

### 1. 扩展的文件类型支持
MarkEdit 现在支持 **50+ 种文件格式**，包括：

| 类别 | 支持格式 | 示例 |
|------|----------|------|
| **纯文本** | `.txt`, `.log`, `.md` | 笔记、日志、文档 |
| **编程语言** | `.js`, `.py`, `.java`, `.cpp` | 代码文件 |
| **Web 开发** | `.html`, `.css`, `.json`, `.xml` | 网页和配置 |
| **配置文件** | `.yaml`, `.toml`, `.ini`, `.env` | 应用配置 |
| **脚本文件** | `.sh`, `.ps1`, `.bat` | 自动化脚本 |
| **数据文件** | `.csv`, `.sql` | 数据和查询 |
| **特殊文件** | `Dockerfile`, `Makefile` | 构建配置 |

### 2. 智能文件类型检测
- 🔍 **自动识别**: 根据扩展名智能识别文件类型
- 🎨 **文件图标**: 每种文件类型都有独特的图标
- 📝 **编辑器适配**: 自动调整编辑器模式

### 3. 简化的非 Markdown 编辑
- 📄 **源码模式**: 非 Markdown 文件使用纯文本编辑
- 🚫 **无分屏渲染**: 保持简洁，不需要复杂预览
- ⚡ **快速加载**: 轻量级处理，性能优秀

```javascript
  // 返回类型信息、图标、编辑器模式等

```

### 编辑器适配逻辑
```javascript
// 根据文件类型调整编辑器
function adjustEditorForFileType(filename) {
  const fileType = getFileType(filename);
  
  if (fileType.mode === 'code' || fileType.mode === 'plain') {
    // 非 Markdown 文件：使用源码模式，禁用分屏渲染
    editor.setMode('sv'); // 源码模式
    disableMarkdownFeatures();
  } else {
    // Markdown 文件：保持完整功能
    enableMarkdownFeatures();
  }
}
```
