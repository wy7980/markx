# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-04-04

### Added
- 🎉 初始发布
- ✨ Tauri 桌面应用框架
- ✍️ Vditor 所见即所得编辑器
- 📝 三种编辑模式（所见即所得/即时渲染/分屏预览）
- 📂 侧边栏文件管理
- 📑 大纲导航
- 🌓 暗色/亮色主题切换
- 💾 本地自动保存
- ⌨️ 快捷键支持
- 🧪 完整测试套件
  - 单元测试（32 个）
  - 组件测试（13 个）
  - E2E 测试（24 个）

### Features
- **编辑器**
  - Markdown 实时渲染
  - 标题、列表、引用、代码块
  - 粗体、斜体、删除线、链接
  - 表格、图片支持

- **文件管理**
  - 新建/切换/删除文档
  - 自动保存
  - 导出 Markdown

- **用户体验**
  - 响应式布局
  - 主题切换
  - 大纲导航
  - 字数统计
